# 📘 Case Study: Malware Scanner Application Using Java

## 1. Introduction

With the rapid growth of digital systems, malware (malicious software) has become a major threat to computer security. Malware includes viruses, worms, trojans, ransomware, and spyware that can damage systems, steal data, or disrupt operations.

This project focuses on developing a **Malware Scanner using Java** that detects suspicious files based on known signatures and behavioral patterns.

---

## 2. Problem Statement

Traditional antivirus solutions may:

* Be expensive
* Require frequent updates
* Consume high system resources

The objective is to build a lightweight malware scanner that:

* Detects malicious files efficiently
* Scans directories and files
* Uses signature-based detection
* Provides real-time feedback to users

---

## 3. Objectives

* To develop a file scanning system using Java
* To identify malware using hash/signature matching
* To create a user-friendly interface
* To ensure fast and efficient scanning

## 6. Methodology

### Step 1: File Selection

* User selects a directory or file to scan

### Step 2: File Reading

* Read file contents using Java IO

### Step 3: Hash Generation

* Generate hash (MD5 / SHA-256) for each file

### Step 4: Signature Matching

* Compare generated hash with known malware signatures

### Step 5: Detection Result

* If match found → Mark as Malware
* Else → Safe file

## 7. Implementation (Sample Code Snippet)

```java
import java.io.*;
import java.security.*;

public class MalwareScanner {

    public static String getFileChecksum(MessageDigest digest, File file) throws IOException {
        FileInputStream fis = new FileInputStream(file);
        byte[] byteArray = new byte[1024];
        int bytesCount;

        while ((bytesCount = fis.read(byteArray)) != -1) {
            digest.update(byteArray, 0, bytesCount);
        }
        fis.close();

        byte[] bytes = digest.digest();

        StringBuilder sb = new StringBuilder();
        for (byte b : bytes) {
            sb.append(String.format("%02x", b));
        }

        return sb.toString();
    }

    public static void main(String[] args) throws Exception {
        File file = new File("test.exe");

        MessageDigest md = MessageDigest.getInstance("SHA-256");
        String checksum = getFileChecksum(md, file);

        // Example malware signature
        String malwareHash = "abc123xyz";

        if (checksum.equals(malwareHash)) {
            System.out.println("⚠ Malware Detected!");
        } else {
            System.out.println("✅ File is Safe");
        }
    }
}
```
